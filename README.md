1. Чем отличаются режимы сканирования с точки зрения сетевого трафика?
Отличия режимов:

SYN — работает на полном рукопожатии, быстрый, точный, но его могут заметить IDS.

FIN/Xmas — используют пакеты, которые не должны появляться без установленного соединения; ответ RST говорит о закрытом порте, отсутствие ответа — об открытом или фильтруемом.

UDP — основан на ICMP-ответах «порт недоступен»; медленный, ненадёжный.


2. Как отвечает сервер Metasploitable?

На SYN: открыт → SYN+ACK; закрыт → RST.

На FIN/Xmas: открыт → (тишина); закрыт → RST.

На UDP: закрыт → ICMP port unreachable; открыт → чаще тишина//
